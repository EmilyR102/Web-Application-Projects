Problem 2:
Type coercion is when Java script is given two types that one would expect wouldn't work (e.g. string plus number) and coerces one into a compatible type that's compatible.

Example 1: 1+"hi" gives the string "1hi" but Python would throw an exception. JS coerced 1 to a string so it could append to "hi".

Example 2: [1,2] + 3 string "1,23" but Python would throw an exception. JS coerced both to string so they can combine.
